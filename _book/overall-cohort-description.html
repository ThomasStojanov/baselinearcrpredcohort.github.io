4.2 Load data

I download the data set coming from the previous section of the previous chapter 2.5.

load("01-results.RData")

I merge the two data sets to prepare my final data set using the full_join() function from the dplyr package.

dataorig <- dplyr::full_join(
  data,
  screening,
  by = "ptn"
)
data <- dataorig |> 
  pivot_wider(
    id_cols = ptn,
    names_from = event,
    names_prefix = "event_",
    values_from = 
      -names(data)[1:2]
  )

Below, I filter the data to have only the patients that were enrolled in the study using the filter() function from the dplyr package:

data <- 
  data %>% dplyr::filter(
  enrolled_event_Baseline == "Enrolled"
)

data <- data %>% 
  mutate(
    hosptype = hosptype.x_event_Baseline
  ) %>% 
  dplyr::select(
    -hosptype.x_event_Baseline,
    -hosptype.y_event_Baseline
  )
