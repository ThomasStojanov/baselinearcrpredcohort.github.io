Chapter 2 Data management

In this section, we will prepare the data for data analysis.

2.1 Load packages

The following packages will be used.

library(haven) # to use as_factor() function
library(tidyverse) # to use the pipe function %>% along with data management tools

2.2 Load data

For this work, we need two data sets:

  • data: for all enrolled patients
  • screening: for all screened patients

We use here the function called read_dta() to call dta files coming from a STATA export.

data <- haven::read_dta("data/arcrpred-02-BaseFU-sp006-data-2024-01-21_1703.dta")
screening <- haven::read_dta("data/arcrpred-02a-screening-2023-01-16_1444.dta")

With the function labelled::is.labelled() and haven::as_factor(), we convert the labelled versions of the variables to factors:

data <- data %>%  #for data set
  dplyr::mutate_if( #transform labelled variables into factors
  labelled::is.labelled,
  haven::as_factor
) %>% 
  dplyr::mutate_if(
    is.double,
    as.numeric
  )
screening <- screening %>%

  labelled::is.labelled,
  haven::as_factor
) %>% 
  dplyr::mutate_if(
    is.double,
    as.numeric
  )

2.3 Variable modification

There are a bunch of variables we need to modify before being analyzed in the next chapters.

2.3.1 Change variable format

We need to modify the values for specific variables so that they become ordinal. Here, we use the across() function in order to select specific variables and apply the same transformation.

2.3.1.1 anspoconf, anspomot, worksat, workmot

data <- data %>% mutate(
  across(c("anspoconf", "anspomot", "worksat", "workmot"),
    ~ as.integer(case_when(
      .x == "Not at all0" ~ "0",
      .x == "Fully10"     ~ "10",
                         TRUE                ~as.character(.x)))
  ))

2.3.1.2 pain1wo

data <- data %>% 
  mutate(
    pain1wo =
      as.integer(case_when(
        pain1wo == "No pain0" ~ "0",
        pain1wo == "Intolerable pain10"     ~ "10",
        TRUE                       ~ as.character(pain1wo)
      ))
  )

2.3.1.3 sleepnrs

data <- data %>%

2.3.2 Variables re-categorization

2.3.2.1 education2

There, we relied on the Swiss Minimal Dataset for Orthopedics to guide our choices.

data <- data %>% 
  dplyr::mutate(
    education2 =
      case_when(
        education %in% c("Compulsory school", "No education") ~ "Up to 9 years of education",
        education %in% c("Apprenticeship", "High school") ~ "10 to 13 years of education",
        education %in% c("Higher professional education", "University, College") ~ "13+ years of education"
      )) %>% 
  dplyr::select(-education)

data$education2 <-
  factor(data$education2, levels = c("Up to 9 years of education", "10 to 13 years of education", "13+ years of education"))

2.3.2.2 start_active_movements_modif

There, we categorize the start_active_movements variable into a 3-category variable.

2.3.2.6 immrifatisprepair

Here, we probably need a bit of an explanation. We would like to have the highest level of fatty infiltration observed across the 3 tendons. This is why we created the following variable: immrifatrepair_modif.

data <- 
  data |> 
  dplyr::mutate(
    immrifatisprepair =
      case_when(
        is.na(sgispsuture) ~ NA,
        TRUE ~ as.character(immrifatisp)),
    immrifatssprepair =
      case_when(
        is.na(sgsspsuture) ~ NA,
        TRUE ~ as.character(immrifatssp)),
    immrifatsscrepair =
      case_when(
        is.na(sgsscsuture) ~ NA,
        TRUE ~ as.character(immrifatssc))) |> 
  tidyr::unite(
    "immrifatrepair",
    immrifatisprepair,
    immrifatssprepair,
    immrifatsscrepair,
    na.rm = T,
    sep = ", ") |> 
  dplyr::mutate(
    immrifatrepair_modif =
      as.factor(case_when(
        str_detect(immrifatrepair, "4") ~ "Stage 3-4",
        str_detect(immrifatrepair, "3") ~ "Stage 3-4",
        str_detect(immrifatrepair, "2") ~ "Stage 2",
        str_detect(immrifatrepair, "1") ~ "Stage 1",
        str_detect(immrifatrepair, "0") ~ "Stage 0"
      )))

2.3.3 analcohol

The number of patients owing to the variable category “Several times a day” was deemed too low. We therefore decided to re-combine the two adjacent categories “1x daily” and “Several times a day”.

data = 
  data |> 
  dplyr::mutate(
    analcohol_modif = 
      as.factor(
        case_when(
          analcohol == "1x daily" ~ "At least daily",
          analcohol == "Several times a day" ~ "At least daily",
          TRUE ~ as.character(analcohol)
        )
      )
  ) 

data$analcohol_modif <-
  factor(data$analcohol_modif, 
         levels = c("No", 
                    "Occasionally",
                    "At least daily"
                    )
         )

2.3.4 asa

The number of patients owing to category “IV” of the ASA Classification was deemed too low. We therefore decided to recombine the two adjacent categories into one.

data =

data$asa_modif <-

2.4 Variable creation

Here, we create a variable distinguishing between private and public hospitals. We need also to tell the variable which clinic owes to which type of hospital. We used the map showed in Figure 2.1 to prepare this variable.

Figure 2.1: Geographical distribution of ARCR_Pred cohort study centers. Note: Yellow and blue text indicate public and private hospital study centers, respectively. Study centers located at the same city are listed alphabetically. 

data <- data %>% 
  mutate(
   hosptype = case_when(
        Clinic %in% c("ART", "END", "GUT", "HUG", "HIR", "INM", "SON", "KWS") ~ "Private",
        TRUE ~ "Public"
    )
  )
screening <- screening %>% 
  mutate(
   hosptype = case_when(
        Clinic %in% c("ART", "END", "GUT", "HUG", "HIR", "INM", "SON", "KWS") ~ "Private",
        TRUE ~ "Public"
    )
  )
